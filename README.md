# 🎵 SoundWave

Müzik streaming platformu. Spotify tarzı.

## Özellikler
- 🎶 Müzik streaming
- 📂 Çalma listeleri
- 🔍 Müzik keşfi
- 💾 Offline dinleme
- 🎧 Podcast desteği
- 📊 Dinleme istatistikleri

## Teknolojiler
- React Native + Web
- Node.js
- MongoDB
- AWS S3

## Kurulum
```bash
npm install
npm run dev
```
